Cancer Gene Index
Templated Searches in this category includes associations between diseases, gene products, and chemical agents.
The following templated searches are available in the Cancer Gene Index category.
Disease to Genes
Returns genes associated with a disease.
Query Option
Description
Disease (name or concept code)
Enter the name or EVS concept code of the disease you want to include in the search results.
Advanced search criteria
Select one or more of the following check boxes to include data of that type in your search results.
•
Include negative associations
•
Include unfinished curation status
•
Include data collected from cell lines
Agent to Genes
Returns genes whose function and/or expression is influenced by the named agent.
Query Option
Description
Agent (name or concept code)
Enter the name or EVS concept code of the agent you want to include in the search results.
Advanced search criteria
Select one or more of the following check boxes to include data of that type in your search results.
•
Include negative associations
•
Include unfinished curation status
•
Include data collected from cell lines
Gene to Diseases/Agents
Returns associated diseases and agents of a given gene name.
Query Option
Description
