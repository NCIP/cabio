Performing a Templated Search
Templated searches are pre-configured searches of caBIO data that are categorized by functionality.
To perform a templated search
1.
Click the Templated Searches tab.
1.
Select a templated search category by clicking it. The category expands to show a drop-down list.
2.
From the drop-down list, select the templated search you want to run. Options applicable to that templated search appear. For more information on the options available for each templated search category, see:
•
Genome Range Queries
•
Microarray Annotations
•
Genomic Annotations
•
Pathways
•
Cancer Gene Index
3.
Specify the options you want to apply to the templated search.
4.
Click Search.
To return all options to their default values, click Reset.
All caBIO data objects matching the options you specified appear in the search results (see below for example search results).
a.
Click a row in the search results to see details (shown below) about the object you selected.
You can navigate between the Templated Searches home page and the search results page by clicking the << Return to templates and << Return to results links in the blue bar above the detailed results.
b.
Below the details, click the link in the External Links section to open a new browser window that shows the object in the caBIO Object Graph Browser. Links in the caBIO Object Graph Browser (shown below) show additional levels of detail about the object and its attributes.
