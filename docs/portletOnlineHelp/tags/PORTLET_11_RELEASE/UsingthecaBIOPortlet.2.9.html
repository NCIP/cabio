Using the caBIO Portlet > Searching caBIO Data > Performing a Templated Search > Pathways
Pathways
Templated searches in this category include information about biological pathways.
The following templated searches are available in the Pathways category.
Pathway By Name
Returns pathways containing a specific name.
Query Option
Description
Pathway Name
Enter the pathway name you want to include in the search results.
Pathway by Gene Symbol
Returns pathways that a specified gene symbol participates in.
Query Option
Description
Gene/HUGO Symbol or Alias
Enter the gene, HUGO symbol, or alias you want to include in the search results.
Pathway by Source
Returns pathways that a specified source participates in.
Query Option
Description
Pathway Source
From the list, select the pathway source you want to include in the search results.
Pathway by Protein
Returns pathways that a specified protein participates in.
Query Option
Description
Protein Name or UniProt Accession
Enter the protein or UniProt accession number that you want to include in the search results.
