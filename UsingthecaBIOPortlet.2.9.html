Pathways
Reports in this category include information about biological pathways.
The following reports are available in the Pathways category.
Pathway By Name
Returns pathways containing a specific name.
Report Option
Description
Pathway Name
Enter the pathway name you want to include in the report.
Pathway by Gene
Returns pathways a specified gene participates in.
Report Option
Description
Gene/HUGO Symbol
Enter the gene or HUGO symbol you want to include in the report.
