Performing a Templated Search
Templated searches provide you with pre-configured searches of caBIO data categorized by functionality.
To perform a templated search
1.
Select a templated search category by clicking it. The category expands to show a drop-down list.
2.
From the drop-down list, select the templated search you want to run. Options applicable to that templated search appear. For more information on the options available for each templated search category, see:
•
Genome Range Queries
•
Microarray Annotations
•
Genomic Annotations
•
Pathways
•
Cancer Gene Index
3.
Specify the options you want to apply to the template search.
To return all options to their default values, click Reset.
4.
Click Search.
All caBIO data objects matching the templated search options you specified appear in a results table in a browser page. The number of available results appears at the top of the results table.
Click the link in each object’s Details column to see more information about that object. Each succeeding details page includes additional links that allow you to drill down into several components of the domain object.
