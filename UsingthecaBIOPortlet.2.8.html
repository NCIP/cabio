Microarray Annotations
Reports in the Microarray Annotations category include manufacturer-provided annotations from the most widely used human microarrays from Affymetrix (http://www.affymetrix.com), Illumina (http://www.illumina.com), and Aglient (http://www.agilent.com).
The following reports are available in the Microarray Annotations category.
Reporter Lookup by Gene
Returns all annotations in all microarrays associated with a specified HUGO gene symbol.
Report Option
Description
Gene Symbol
Enter the gene or HUGO symbol you want to include in the report.
Reporter Lookup by SNP
Returns all annotations in all microarrays associated with a specified dbSNP identifier.
Report Option
Description
dbSNP Id
Enter the dbSNP Id you want to include in the report.
Reporter Lookup by Name
Returns annotations for a specified reporter name in a specified microarray.
Report Option
Description
Microarray
Enter the name of the microarray you want to include in the report.
Reporter Name
Enter the reporter name you want to include in the report.
