To download a Microsoft Excel spreadsheet with this browser, follow these steps:
  1. Press the "control" key.
  2. While pressing the control key, click on the "HERE" link.
  3. Select the "Download Link to Disk" option from the popup menu.
  4. Enter a sensible name for your spreadsheet in the "Save As" box of the "Save" dialog. Be sure to name the file with the ".xls" extension.
  5. Save the file somewhere on you computer.

To view your spreadsheet, open Microsoft Excel (first) and then navigate to your saved file.
