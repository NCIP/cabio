Welcome to BIOgopher!!!

BIOgopher is a bioinformatics tool that facilitates the integration of your local bioinformatics data with public data sources via cancer Bioinformatics Infrastructure Objects (caBIO) at the NCI Center for Bioinformatics.
Using BIOgopher you can:
  • Map your local data in an Excel spreadsheet to caBIO data objects
  • Create a query based on search criteria that can include a column from your spreadsheet
  • Create a final report that delivers the results of your query in another Excel spreadsheet, with your original data merged with the results
You can perform all of these activities without doing any programming. If you do not have any local spreadsheet data, you can still browse and query the caBIO objects and create reports of caBIO data. For more information on caBIO and its contents, please visit http://ncicb.nci.nih.gov/core/caBIO.
