Creating Reports

Once you have created a query, you can create a report that is based on that query. The report is based on your query in that each row of your report will describe the object that you initially selected when creating your query.

On this page, you can either create a new report or work with an existing one. To create a new report, select the query on which it will be based from the drop-down list in step A, on the left, under Create A New Report. The give your new report a name. Finally, press create. You'll then see an interface for designing your report.

To work with an existing report, select the name of the report from the drop-down list in step A, on the left, under Work With An Existing Report. Then you can edit, rename, or remove that report from your session.
