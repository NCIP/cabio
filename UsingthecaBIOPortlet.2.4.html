Performing a Simple Search
To perform a simple search in the caBIO Portlet
1.
Click the Simple Search tab.
2.
Enter a search word or words into the text box. Word that exist in the caBIO domain and will therefore result in matches are suggested in a list box. Click one of the suggestions to select it.
3.
Optionally, click the Advanced Options link and make your search results more specific by selecting any of the following options:
Advanced Search Options
Explanation
Match: any
Match any of the words in the text box.
Match: all
Match all of the words in the text box.
Exclude
Enter any terms that should not appear in the search results.
View: simple
Show a title, description, and caBIO class for each search result.
View: objects
Show all object information for each search result. This query is slower but may return more results (for example, Grid IDs).
Results per page
Select 5, 10, or 15 results per page.
4.
Click Search. A list of caBIO objects matching the search criteria appear. Search results include a title, description, and caBIO class.

You can also filter your results to a single caBIO class by clicking a result category (see illustration below). Result categories appear in the blue and yellow box that also contains the total number of results. An example appears below.
In this example, the search was for any instance of the word “glaucoma,” which resulted in 48 records. You can then sort those 48 records by object class such as Evidence, Gene, Protein, and Agent. In this example, 28 of the results apply to the Evidence object class, 15 to Gene, 4 to Protein, and 1 to Agent. Click a result category to see only those results from the selected caBIO class.
5.
Click an object name in the search results to see details about that object, as shown below.
6.
You can do the following to continue browsing caBIO data.
•
Click the red arrow to the right of a row to get more information about the object. Additional detail may include gene, disease, and agent associations, links to PubMed, and any other information that may be associated with the object. An example of a detailed search page appears below. Continue clicking the red arrows to further browse the object.
•
Click the link in the External Links section below the object’s details to see the object in the caBIO Object Graph Browser, shown below. Links in the caBIO Object Graph Browser show additional levels of detail about the object and its attributes.
