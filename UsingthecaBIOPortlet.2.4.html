Performing a Simple Search
The caBIO Portlet provides a simple search for performing “Google-like” queries in caBIO. Using this search facility, users can search caBIO by entering one or more keywords. Since caBIO provides a model representation and repository of molecular biology data, entities that concern the Central Dogma of Molecular Biology (Genes, Proteins, Nucleic Acid Sequences, Chromosomes) are the core of this model and the primary subject for search terms. The results of the simple search are formatted based on the domain object represented in the caBIO model.
The simple search is based on the caBIO Freestyle Lexical Mine (FreestyleLM) search that builds upon Hibernate and Apache Lucene full text search engines. The FreestyleLM search facilitates a horizontal search across caBIO domain objects. For example, searching for “brca1” will return any object that includes the brca1 gene in its attributes, such as Genes, Proteins, Pathways, and so on. Hibernate searches are object-based (a "vertical" search). In a vertical search, you must select one caBIO domain object (such as Gene) to search.
To perform a simple search in the caBIO Portlet
1.
Click the Simple Search tab.
2.
Enter a search term or terms into the text box. Search terms must exist in the caBIO model.
3.
Optionally, click the Advanced Options link and filter your search results by selecting any of the following options:
Advanced Search Options
Explanation
View: simple
Displays only the most relevant annotations in the search results for performance efficiency. Formatting is applied to each result, based on the caBIO domain object type, to show a relevant title and description and class for each result.
View: objects
Displays all information associated with each caBIO object in the search results. After performing the simple search, an additional search is performed to retrieve the full object for each result. The query will be slower as more data is returned.
Match Terms: any
Match any of the words in the text box.
Match Terms: all
Match all of the words in the text box.
Exclude
Enter any terms that should not appear in the search results.
Note: caBIO data that contain the search term are excluded in the search results; however, caBIO fields in the database named after the excluded term will not be excluded as the fields may contain relevant data.
4.
Click Search. A list of caBIO domain objects matching the search criteria appear. Search results include a title, description, and caBIO class.
5.
Click an object name in the search results to see details about that object.
