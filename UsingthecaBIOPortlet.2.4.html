Performing a Simple Search
To perform a simple search in the caBIO Portlet
1.
Click the Simple Search tab.
2.
Enter a search term or terms into the text box. Search terms must exist in the caBIO domain.
3.
Optionally, click the Advanced Options link and filter your search results by selecting any of the following options:
Advanced Search Options
Explanation
Match: any
Match any of the words in the text box.
March: all
Match all of the words in the text box.
Exclude
Enter any terms that should not appear in the search results.
Domain-specific formatting
Formatting will be applied to each result, based on the object type, to show a relevant title and description and class for each result.
Map to objects (slower)
After performing the Lucene search, performs an additional Hibernate-based search to get back the full object for each result. Thus, the query will be slower, but may return more data (for example, Grid Ids). Note that domain-specific formatting must be turned OFF to see these additional data.
4.
Click Search. A list of caBIO domain objects matching the search criteria appear. Search results include a title, description, and caBIO class.
5.
Click an object name in the search results to see details about that object.
