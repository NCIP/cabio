Cancer Gene Index
Reports in this category includes associations between diseases, gene products, and chemical agents.
The following reports are available in the Cancer Gene Index category.
Disease to Genes
Returns associated genes given a disease name.
Report Option
Description
Disease (name or concept code)
Enter the name or EVS concept code of the disease you want to include in the report.
Agent to Genes
Returns genes whose function and/or expression is influenced by the named agent.
Report Option
Description
Agent (name or concept code)
Enter the name or EVS concept code of the agent you want to include in the report.
Gene to Diseases/Agents
Returns associated diseases and agents of a given gene name.
Report Option
Description
Gene Symbol
Enter the gene or HUGO symbol you want to include in the report.
