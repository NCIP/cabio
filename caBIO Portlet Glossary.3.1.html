caBIO Portlet Glossary
Acronyms, objects, tools and other terms related to the caBIO Portlet are described in this glossary.
Term
Definition
annotation
Information about genomic objects, such as position of intron-exon-boundaries, regulatory sequences, repeats, gene names and protein products, and their function e.g. Gene Ontology.
agent
A therapeutic agent (drug, intervention therapy) used in a clinical trail protocol.
assembly
A particular version of a genome, assembled from short DNA sequences generated by shotgun sequencing.
CBIIT
NCI Center for Biomedical Informatics and Information Technology - see http://ncicb.nci.nih.gov/.
chromosome
Part of a cell that contains genetic information. Except for sperm and eggs, all human cells contain 46 chromosomes.
clone
An object used to hold information pertaining to I.M.A.G.E. clones; provides access to sequence information, associated trace files, and the clone's library.
dbSNP identifier
The identifier for a cluster of polymorphisms in dbSNP, NCBI’s central repository for single base nucleotide substitutions (SNPs) and short deletion and insertion polymorphisms (aka “rs number”) - see http://www.ncbi.nlm.nih.gov/SNP.
downstream distance
Relative distance between the SNP and the genomic feature measured in base pairs, or centimorgans, e.g. Downstream (or 3').
end position
The nucleotide number (starting from the 5 end of the chromosome) at which the feature (gene, SNP, etc.) ends.
EVS
Enterprise Vocabulary Services. For more information, see http://ncicb.nci.nih.gov/NCICB/infrastructure/cacore_overview/vocabulary.
gene
The functional and physical unit of heredity passed from parent to offspring. Genes are pieces of DNA, and most genes contain the information for making a specific protein.
genome
The complete sequence of DNA contained in an organism or a cell, including both the chromosomes within the nucleus and the DNA in mitochondria.
HUGO gene symbol
A gene symbol approved by and included in the HGNC (HUGO Gene Nomenclature Committee database) - see http://www.gene.ucl.ac.uk/nomenclature.
library
An object representing a CGAP library; provides access to information about: the tissue sample and its method of preparation, the library protocol that was used, the clones contained in the library, and the sequence information derived from the library.
microarray
A process that allows thousands of pieces of DNA that are fixed to a glass slide to be analyzed at one time. It is used to identify the genes (pieces of DNA) in specific cells or tissue that are actively used to make RNA, which then may be used to make proteins.
NCBI
National Center for Biotechnology Information - see http://www.ncbi.nlm.nih.gov/.
pathway
An object representation of a molecular/cellular pathway compiled by BioCarta. Pathways are associated with specific Taxon objects, and contain multiple Gene objects, which may be Targets for treatment.
reporter
rA generic object to represent features on a particular array platform for measuring specific biological signals, such as genotype for SNP array, and levels of gene transcription products for Expression and Exon arrays.
sequence
The sequence of nucleotide residues along a DNA chain.
SNP
Single nucleotide polymorphism: A SNP occurs when corresponding sequences of DNA from different individuals differ at one DNA base; for example, where the sequence AAGCCTA changes to AAGCTTA.
start position
The nucleotide number (starting from the 5 end of the chromosome) at which the feature (gene, SNP, etc.) begins.
UniProt accession number
A unique identifier given to a DNA or protein sequence record to allow for tracking of different versions of that sequence record and the associated sequence over time in a single data repository.
upstream distance
Relative distance between the SNP and the genomic feature measured in base pairs, or centimorgans, e.g. Upstream (or 5').
