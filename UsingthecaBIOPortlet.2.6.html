Genome Range Queries
The templated searches in this category include data on genomic features such as genes, markers, SNP, and probe_set within a specified chromosomal region.
The following templated searches are available in the Genome Range Queries category.
Absolute Range Query
Returns specified genomic features for any object that has a Physical Location, based on its start and end position.
Query Option
Description
Species
Select a species from the drop-down list.
Chromosome
Select the chromosome numbers associated with the selected species.
Assembly
Select an assembly.
Start Position
Enter a start position.
End Position
Enter an end position.
Display
Select which genomic feature class to display in the search results.
Grid Id Range Query
Returns specified genomic features for any object that has a Physical Location, based on its caBIO grid identifier.
Query Option
Description
Grid ID
Enter the identifier for a genomic feature
Downstream Distance
Enter a downstream distance.
Upstream Distance
Enter an upstream distance.
Display
Select which genomic feature class to display in the search results.
